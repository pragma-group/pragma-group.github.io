N2284=07-0144, Working Draft, Standard for Programming Language C++, contains the working draft for the standard, with all the changes approved at the Oxford meeting other than LWG Issue 258.

Change bars have not been consistently applied in code. It should be clear from context where new code chunks have been added. In particular, if ‘Example’ and ‘end example’ are marked as added, the code between them has also been added.
