New Papers

  • V1 of the Library Fundamentals TS has been sent for publication.
  • N4529 is the current working draft for V2 of the Library Fundamentals TS. It replaces N4336.
  • N4530 is this Editor's Report.

V1 has 3 issues applied from the Lenexa meeting, and is being sent for publication.

V2 has 5 motions applied from the Lenexa meeting.

Notable Changes to Issues and Papers as Moved

Retargeted N4371 to the Fundamentals TS

This paper was written as changes to the main C++ standard, so I had to adjust headers and section numbers to make it fit into the TS.

Updated the feature testing macros

Not all applied papers proposed feature-test macros, so I invented some. I also gave all new features a value of 201505.

Rearranged the Source Locations paper a bit

Put definitions outside the synopsis section, gave the synopsis a "synop" stable name, and put all creation operations together.
